# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

We take security seriously. We run a **Bug Bounty Program** to encourage responsible disclosure.

**To participate in the bug bounty:**

1. ⭐ **Star this repository** — All bounty participants must have starred the repo
2. Open a security issue using the Bug Bounty template
3. Provide a detailed description of the vulnerability
4. Include steps to reproduce
5. Submit a PoC if possible

**Bounty rewards:**

| Severity   | Reward        |
| ---------- | ------------- |
| Critical   | $5,000 - $10,000 |
| High       | $2,000 - $5,000  |
| Medium     | $500 - $2,000    |
| Low        | $100 - $500      |

We typically respond within 24 hours and resolve critical issues within 72 hours.
